# рџ’» Local Setup Steps

## 1. Clone Repository

```bash
git clone https://github.com/Chiragverma0786/Project-Management-Tool.git

cd Project-Management-Tool
```

---

## 2. Backend Setup

```bash
cd Backend

npm install
```

Create a `.env` file inside the Backend folder.

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_CONNECTION_STRING
JWT_SECRET=YOUR_SECRET_KEY
NODE_ENV=development
```

Run backend:

```bash
npm run dev
```

Backend runs on:

```
http://localhost:5000
```

---

## 3. Frontend Setup

```bash
cd Frontend

npm install
```

Create `.env.development`

```env
VITE_API_URL=http://localhost:5000/api
```

Run frontend

```bash
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

# рџ”ђ Authentication

The application uses JWT (JSON Web Token) authentication.

After a successful login:

- JWT token is generated by the backend.
- Token is stored in Local Storage.
- Axios Interceptor automatically attaches the token to every protected API request.
- Unauthorized users are redirected to the login page.
